A simple Node.js + Express web app that fetches random cocktail recipes
from TheCocktailDB API. Users can add cocktails to a cart and manage them.

## Tech Stack
- Node.js
- Express
- EJS
- PostgreSQL
- TheCocktailDB API

## How to Run
1. npm install
2. node index.js
3. Open http://localhost:3100
